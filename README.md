# Experimental Study of Two Coupled Pendulums using FFT and Mobile Sensors

This project presents an experimental analysis of two coupled pendulums using mobile sensor data and Fast Fourier Transform (FFT) techniques to identify oscillation modes and frequency behavior.

## Objectives

- Analyze the oscillatory behavior of coupled pendulums
- Extract dominant frequencies using FFT
- Compare normal modes experimentally
- Visualize experimental data with Power BI

## Tools and Technologies

- Python
- NumPy
- Matplotlib
- Power BI
- Mobile sensor data
